## Chat

## Описание

Веб-версия чата на нативном JS.
Используемые технолонии:

- `Pug`
- `Scss`
- `JS`
- `Parcel`
- `Node.js`
- `Express`

## Установка

- `npm i` — установка проекта,
- `npm start` — запуск версии для разработчика,
- `npm run build` — сборка стабильной версии.
